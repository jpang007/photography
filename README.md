# Jeremy Pang Photography

Personal photography portfolio showcasing amateur photography from Seattle.

## About

This is a personal website built to share photos taken for fun. The site features work across four main categories:

- **Street** - Urban life and candid moments
- **Travel** - Places and cultures
- **Landscape** - Nature and scenery
- **Architecture** - Buildings and structures

## Tech Stack

Built with modern web technologies:

- **Next.js 13** - React framework
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Responsive Design** - Mobile, tablet, and desktop optimized

## Features

- Clean, minimal design focused on photography
- Interactive lightbox for full-screen viewing
- Category-based filtering
- Fully responsive across all devices
- Optimized image loading

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view locally.

## Deployment

This site can be deployed to:
- **Vercel** (recommended for Next.js)
- **Netlify**
- Any Node.js hosting platform

## Content Management

All site content, photos, and configuration are managed in `src/data/siteData.ts`. Photos are stored in the `public/images/` directory.
